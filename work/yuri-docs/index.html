注意事项 #

使用前须知：

  • 系统下户设定起充500$，需先打款

  • 死户需要等7天回流期结束才可以让我方工作人员来操作清零归档

  • 第一次打款需要单独转款，之后可自行使用系统自助充值

  • 下戶会有考核率，空户率或者封户率过高会限制下户

  • 下户需求请提前在群组提，正常情况下当天提可以当天下户

工作时间：每日8:30-24:00

中场时间: 11:25-12:45,17:30-19:00

工作时间以外也可以在群里提需求，但工作时间以外请勿电话催促或者催 单，工作人员看到也会处理，或者第二天处理。

谢谢理解与配合。

如何注册尤里改团队 #

  1. 打开后台，选择注册账号

  2. 填入注册信息

  3. 左上角是团队名字，请发给工作人员激活
