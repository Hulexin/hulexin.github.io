关于非遗出海

思来想去还是想做非遗出海，没有高大上的文化交流，只是搞钱加上喜欢，非遗有内容好做内容，产品赋值高，文化的差异容易带给客户高情绪价值。目前应该系统上会支持这类商业行为，也有很多需要展示渠道的非遗人。

愿景 #

打造最大的非遗电商网站

核心产品与服务 #

1. 非遗集成销售平台 #

  • 建立独立站（多语言、多币种）展示各类非遗商品，如手工刺绣、传统陶瓷、剪纸、竹编、传统服饰等。
  • 对接优质非遗传承人/品牌，提供线上展销渠道。
  • 提供物流、客服、订单管理等一站式运营支持。
  • 设置“故事”版块，展示非遗背后的文化故事，增强用户黏性和信任。

2. 非遗品牌独立站定制服务 #

  • 为希望打造自有品牌的非遗传承人提供建站服务：
    • 网站搭建（Shopify/Shopline/Wix等）
    • 视觉设计与品牌包装
    • 多语言内容撰写与翻译
    • 跨境收款与物流对接
    • 教学培训与售后支持

3. 自营非遗品牌孵化 #

这个是目前构想的主要目的，可以通过1和2来沉淀过滤优质项目，不过落地之前不会知道哪个更有潜力，前面肯定要做好，后面才能更好。

  • 1和2会产出产品数据，内容，素材等，筛选市场潜力高的非遗产品，打造自有品牌。
  • 独立站运营 + TikTok/Youtube/Instagram内容种草 + 品牌故事驱动销售。

目标市场 #

1. 消费人群 #

  • 海外中高收入人群，注重手工艺、独特性与文化价值。
  • 海外华人，有情感认同和文化连接需求。
  • 风水、转运、辟邪等需求人群。
  • 家居、美学、收藏类消费群体。

2. 国家/地区 #

  • 北美（美国、加拿大）
  • 欧洲（德国、法国、英国）
  • 中东高消费市场（迪拜、卡塔尔）

收益来源 #

非遗集成平台 - 抽出佣金 + 平台月费（这个前期估计是免了）

建站服务 - 一次性建站费用 + 运维费用

自有品牌 - 钱来来来！
