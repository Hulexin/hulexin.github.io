2025-08-1 #

敏感的柚子 #

可能是看到别人玩的车子，想到了家里的车子，可能是其他的原因让佑宝想之前的家了，有些愧疚，说不出来的感觉，我听到睡觉的时候柚子在跟妈妈说那边家的事情，心里堵堵的。

我也开始早上8点多上班，晚上快6点回家，柚子应该也是跟我一样想我吧。好好努力，给我姑娘更好的生活。

2025-07-21 #

搬家了，新征程 #

来跟我哥混了，一切都很顺利，佑宝和夸夸都玩的很开心。安顿下来搞钱。

来到这边，柚子跟姐姐一起更开朗了一些，话多了好多倍哈哈哈，我说为啥我要把家里的两个挖耳刀都带着呢。

2025-07-07 #

我佑崽有了自己的手机号了 #

好巧，选号里面找到了一个53173517 跟我和妈妈尾号一样。

2025-07-02 #

情商开始占领高地 #

早上她姥姥问：是爸爸好还是妈妈好，佑佑回答说：姥姥好。

我觉得这个问题应该是错误的提问，把爸爸和妈妈对立，我忍。

2025-06-20 #

叠叠乐 #

两岁是十一个月的宝宝，竟然能玩叠叠乐了，而且技术还不一般！专业的。很多次都能和妈妈抽空全部。

2025-06-17 #

有姑娘在，睡的真香 #

好久没有和姑娘一个床上睡午觉了，忽然让我跟他一起睡午觉，没两分钟我就睡着了，姑娘也不知道啥时候握着我的手睡着了哈哈哈。幸福的一天儿。

洗头 唱歌 #

今天给姑娘洗头的时候，和姑娘一起唱歌，小星星，还有曾克白有追（一个英文歌，姑娘扭扭车里面的哈哈哈，我们也不知道名字叫啥），然后我给姑娘唱古诗，悯农，春晓，然后姑娘忽然想起来之前教姑娘的鹅鹅鹅，但是姑娘忘记是鹅了，姑娘说鸭鸭鸭，哈哈哈哈，然后就开始乐，笑的整个洗头都没停下。
